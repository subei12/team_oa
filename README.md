# team_oa
为[葫芦侠社区](http://www.huluxia.com/)技术分享板块提供的团队管理系统。

使用地址：https://oa.jsls9.top

本仓库为后端仓库，前端仓库为[team_oa_web](https://github.com/subei12/team_oa_web)

### 技术栈
springboot、shiro、mybatis等

### 前端
使用百度开源项目：[AMIS](https://github.com/baidu/amis)

主要功能：

1. 帖子管理（结算）
2. 我的信息
3. 团队人员管理
4. 积分兑换
5. 团队预算管理
6. 上报管理
